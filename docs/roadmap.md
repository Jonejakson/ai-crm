## Pulse CRM – дорожная карта

Основано на сравнении с amoCRM, Битрикс24 и SMB‑конкурентами (Pipedrive, HubSpot Starter и др.).

### 1. Коммуникации и омниканал
- Подключение телефонии (SIP, запись разговоров, распределение звонков).
- Интеграции с мессенджерами (Telegram, WhatsApp, Viber, соцсети).
- Единая лента диалогов: e‑mail, чаты, звонки в одном окне.

### 2. Автоматизации
- Визуальный конструктор сценариев «если/то» с drag-and-drop.
- Библиотека готовых автоворонок (follow-up, возвраты, напоминания).
- Новые триггеры и действия (платежи, файлы, формы, вебхуки).

### 3. Дашборды и аналитика
- Конструктор отчётов с фильтрами, экспортом, план/факт.
- KPI и performance‑дашборды по менеджерам.
- Прогнозирование выручки и здоровья воронки.

### 4. Документы и продажи
- Счета, КП, договоры, печатные формы и шаблоны.
- Учёт товаров/услуг, интеграция с ЭДО (Диадок, СБИС).
- Связка с YooKassa и другими PSP для автоматических оплат по счетам.

### 5. Маркетинг и лидогенерация
- Конструктор веб-форм и виджетов сайта.
- Email sequences / drip campaigns, планировщик рассылок.
- Мини‑лендинги или интеграции с внешними конструкторами.

### 6. Интеграции и API
- Коннекторы к Google/Outlook, Slack/Teams, 1С / МойСклад.
- Публичный API и конструктор вебхуков.
- Marketplace / SDK для партнёрских модулей.

### 7. Командная работа (опционально)
- Общая лента компании, объявления, быстрые файлы.
- Базовый файловый менеджер, интеграция с Google Drive.
- Простые HR‑процессы (отпуска, расписания).

### 8. Мобильность
- PWA с push‑уведомлениями.
- План нативных приложений (React Native/Flutter) после PWA.

### 9. AI‑ассистент
- Возврат помощника в формате генерации писем, подсказок по сделкам.
- Анализ звонков/чатов, советы по следующему действию.

> Приоритизация: начинать с п.1–4, затем развивать маркетинг, интеграции и AI. Мобильность и командный блок — после закрытия основных сценариев CRM/продаж.
