# Автоматическое подтверждение оплаты по счёту (Т-Банк)

Если вы самозанятый и принимаете оплату по счёту на карту/счёт Т-Банка, можно настроить **автоматическое** подтверждение оплаты в CRM без ручного нажатия «Подтвердить».

## Как это работает

1. Клиент выбирает «Оплата по счёту» → в CRM создаётся счёт (Invoice) в статусе PENDING и PDF с реквизитами.
2. Клиент переводит деньги на вашу карту/счёт Т-Банка (в назначении платежа лучше указать **номер счёта**, например `INV-20250210-0001`).
3. Т-Банк при поступлении денег отправляет уведомление (вебхук) на наш сервер.
4. CRM получает вебхук, находит счёт по сумме и/или номеру в назначении платежа, помечает его как оплаченный и активирует подписку.

## Что нужно сделать

### 1. Подключить вебхук «Операция по счёту» в Т-Банке

- Зайти на [портал разработчика Т-Банка](https://developer.tbank.ru/docs/intro/webhooks).
- Написать на **openapi@tbank.ru** с почты, привязанной к вашему аккаунту в банке. В письме указать:
  - **ИНН** вашей компании/самозанятого;
  - **Вебхук:** «Операция по счёту»;
  - **Адрес:** `https://ваш-домен.ru/api/billing/webhook/tbank` (для Flame CRM: `https://flamecrm.ru/api/billing/webhook/tbank`).
- Подключение обычно занимает до 1–2 рабочих дней.

### 2. Указать номер счёта в назначении платежа

В шаблоне счёта (PDF) или в инструкции клиенту нужно просить указывать **номер счёта** в назначении платежа (например: `Оплата по счёту INV-20250210-0001`). Тогда CRM сможет однозначно сопоставить поступление с нужным счётом. Если номер не указывать, сопоставление идёт только по сумме (возможны совпадения с другими платежами).

### 3. Ограничения

- Вебхук «Операция по счёту» в Т-Банке доступен для **расчётных счетов** (юрлиц/ИП). Для **карты самозанятого** нужно уточнить в Т-Банке (поддержка, openapi@tbank.ru), доступен ли для вашего продукта вебхук или только выписка по API.
- Если вебхук для вашего счёта/карты недоступен, возможен вариант с **периодической проверкой выписки** по API Т-Банка (раз в N минут искать новые входящие операции и сопоставлять со счетами). Для этого нужен токен Т-API с доступом «Счета и выписки».

## Проверка

После подключения вебхука при поступлении оплаты по счёту подписка в CRM должна активироваться автоматически, без ручного подтверждения.
